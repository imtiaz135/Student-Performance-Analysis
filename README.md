# Ecommerce Sales Analysis

This project performs an end-to-end data analysis on e-commerce sales data
using Python.

## Folder Structure
- data/ : Dataset
- visualizations/ : Generated charts
- report/ : Written insights
- main.py : Analysis script

## How to Run
1. Install dependencies  
   pip install -r requirements.txt

2. Run the project  
   python main.py

## Output
- Monthly sales trend
- Top products by revenue
- Sales distribution by region
